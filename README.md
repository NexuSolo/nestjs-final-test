
# Projet Nest

Projet de démonstration Nest.js

## Prérequis

- Installer et lancer Docker Desktop
- Installer node
## Installation

Initialiser les modules du projet avec :

```bash
  npm ci
```

    
## Lancement

Lancer le serveur

```bash
  npm run start:postgres
```


## Lancement des tests

Lancer les tests

```bash
  npm run test:e2e:postgres
```
